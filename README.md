# PrettyDialog
优美悦目的Dialog

特点
=========================
- 高斯模糊背景
- 内置Dialog,方便直接使用
    - ActionSheetDialog
    - MaterialDialog
    - NormalDialog
    - NormalListDialog
- 内置丰富的动画库(包括Dialog动画、Windows动画)
- 支持自定义Dialog、Popup
- 支持自定义Dialog动画、Windows动画
